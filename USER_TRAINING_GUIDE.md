# 用户培训指南

## 培训概述

### 培训目标
- 掌握系统基本操作
- 了解核心功能使用
- 熟悉常见问题处理
- 提升工作效率

### 培训对象
- 系统管理员
- 运维工程师
- 普通用户

### 培训时长
- 基础培训: 2 小时
- 高级培训: 4 小时
- 实操练习: 2 小时

---

## 第一部分：系统登录和基础操作

### 1.1 系统登录
**操作步骤**:
1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 首次登录建议修改密码

**注意事项**:
- 密码区分大小写
- 建议使用强密码
- 定期更换密码
- 不要共享账号

### 1.2 界面介绍
**主要区域**:
- 顶部导航栏: 用户信息、退出登录
- 左侧菜单栏: 功能模块导航
- 主内容区: 功能操作界面
- 面包屑导航: 当前位置

### 1.3 个人信息管理
**修改个人信息**:
1. 点击右上角用户头像
2. 选择"个人信息"
3. 修改姓名、邮箱等信息
4. 点击"保存"

**修改密码**:
1. 进入个人信息页面
2. 点击"修改密码"
3. 输入旧密码和新密码
4. 确认保存

**头像管理**:
1. 点击头像编辑按钮
2. 选择喜欢的头像风格
3. 调整颜色和配置
4. 保存头像设置

---

## 第二部分：用户和权限管理

### 2.1 用户管理（管理员）
**添加用户**:
1. 进入"用户管理"页面
2. 点击"新增用户"按钮
3. 填写用户信息
4. 分配角色
5. 保存用户

**编辑用户**:
1. 在用户列表中找到目标用户
2. 点击"编辑"按钮
3. 修改用户信息
4. 保存更改

**删除用户**:
1. 选择要删除的用户
2. 点击"删除"按钮
3. 确认删除操作

### 2.2 角色管理（管理员）
**创建角色**:
1. 进入"角色管理"页面
2. 点击"新增角色"
3. 填写角色名称和描述
4. 配置菜单权限
5. 保存角色

**权限配置**:
1. 选择角色
2. 勾选允许访问的菜单
3. 保存权限配置

---

## 第三部分：主机运维管理

### 3.1 添加 SSH 主机
**操作步骤**:
1. 进入"主机管理"页面
2. 点击"添加主机"
3. 填写主机信息:
   - 主机名称
   - IP 地址
   - SSH 端口
   - 用户名
   - 认证方式（密码/密钥）
4. 测试连接
5. 保存主机

**认证方式选择**:
- 密码认证: 输入 SSH 密码
- 密钥认证: 上传私钥文件

### 3.2 查看主机信息
**系统信息**:
1. 点击主机详情
2. 查看操作系统版本
3. 查看硬件配置
4. 查看网络信息

**性能监控**:
1. 查看 CPU 使用率
2. 查看内存使用率
3. 查看磁盘使用率
4. 查看网络流量

### 3.3 使用 WebShell
**打开终端**:
1. 在主机列表中点击"打开终端"
2. 等待连接建立
3. 在终端中输入命令
4. 查看执行结果

**常用命令**:
```bash
ls -la          # 查看文件列表
pwd             # 查看当前目录
top             # 查看进程
df -h           # 查看磁盘使用
free -h         # 查看内存使用
```

### 3.4 Ansible 自动化
**创建 Playbook**:
1. 进入"Ansible 管理"
2. 点击"新增 Playbook"
3. 编写 YAML 配置
4. 保存 Playbook

**执行 Playbook**:
1. 选择 Playbook
2. 选择目标主机
3. 配置执行参数
4. 点击"执行"
5. 查看执行日志

## 第四部分：监控告警

### 4.1 配置告警渠道
**邮箱告警**:
1. 进入"告警渠道"页面
2. 点击"新增渠道"
3. 选择"邮箱"类型
4. 配置 SMTP 信息
5. 添加收件人
6. 测试渠道
7. 保存配置

**钉钉告警**:
1. 创建钉钉机器人
2. 获取 Webhook URL
3. 在系统中添加钉钉渠道
4. 配置 Webhook
5. 测试发送
6. 保存配置

### 4.2 配置告警规则
**创建规则**:
1. 进入"告警规则"页面
2. 点击"新增规则"
3. 配置监控指标
4. 设置告警条件
5. 选择告警渠道
6. 启用规则

**规则示例**:
- CPU 使用率 > 80%
- 内存使用率 > 90%
- 磁盘使用率 > 85%
- 负载 > 10

### 4.3 处理告警
**查看告警**:
1. 进入"告警历史"
2. 查看告警列表
3. 筛选告警记录
4. 查看告警详情

**处理告警**:
1. 确认告警
2. 记录处理过程
3. 解决问题
4. 标记为已处理

---

## 第五部分：网络探测

### 5.1 创建探测分组
**操作步骤**:
1. 进入"分组管理"
2. 点击"新增分组"
3. 填写分组名称
4. 设置颜色标识
5. 保存分组

### 5.2 配置网络探测
**HTTP/HTTPS 探测**:
1. 进入"探测管理"
2. 点击"新增探测"
3. 选择协议类型
4. 填写目标 URL
5. 配置请求参数
6. 设置探测间隔
7. 选择分组
8. 保存配置

**配置示例**:
- 协议: HTTPS
- URL: https://www.example.com
- 方法: GET
- 超时: 30 秒
- 间隔: 60 秒

### 5.3 查看探测结果
**实时监控**:
1. 进入"探测监控"
2. 查看实时状态
3. 查看响应时间
4. 查看成功率

**历史数据**:
1. 进入"探测结果"
2. 选择时间范围
3. 查看趋势图表
4. 导出数据

## 第六部分：常见问题

### 6.1 登录问题
**Q: 忘记密码怎么办？**
A: 联系管理员重置密码

**Q: 登录后页面空白？**
A: 清除浏览器缓存，重新登录

### 6.2 主机连接问题
**Q: SSH 连接失败？**
A: 检查网络连接、防火墙设置、SSH 服务状态

**Q: WebShell 无法打开？**
A: 检查 WebSocket 连接，刷新页面重试

### 6.3 告警问题
**Q: 没有收到告警通知？**
A: 检查告警渠道配置、规则是否启用

**Q: 告警太频繁？**
A: 调整告警阈值、增加静默期
